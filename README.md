# Railway Reservation System
This is a console-based Railway Reservation System developed using C++. This project was created as part of the Object-Oriented Programming (OOP) course.
Features
Manage train ticket bookings.
Store and retrieve passenger data.
User-friendly console interface.
How to Run
Ensure you have a C++ compiler installed (like Dev-C++ or MinGW).
Clone this repository or download the source code.
Open the main project file in your IDE.
Compile and run the code.
Built With
Language: C++
Concepts Used: Object-Oriented Programming, File Handling.
